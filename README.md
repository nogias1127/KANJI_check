# 漢字チェック Webアプリ試作

iPad + Apple Pencilで漢字を書く練習用のWebアプリです。
方式A（ルール判定型）として、画数・枠内配置・中心・大きさ・お手本との重なりを見て簡易判定します。

## 使い方

1. `index.html` をブラウザで開く
2. iPadではSafariで開く
3. マスにApple Pencilまたは指で書く
4. 「判定する」を押す
5. 「漢字リスト」から対象漢字を差し替える

## GitHub Pagesに置く場合

このフォルダの中身をリポジトリ直下に置けば、そのまま公開できます。

## 漢字リスト形式

1行ごとに以下の形式で入力します。

```txt
漢字,画数
一,1
二,2
漢,13
```

画数が不明な場合は漢字だけでも動きます。

```txt
漢
字
練
習
```

## 判定内容

- 画数：ペンを離した回数を画数として計算
- 中心：マスの中央からのズレ
- 大きさ：お手本文字とのサイズ差
- 枠内：マス端に寄りすぎていないか
- 形：お手本フォントとの重なり率
